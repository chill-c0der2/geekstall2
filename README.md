Installs Hacking tools KALI:

[+] sudo apt update

[+] sudo apt upgrade

[+] sudo apt install git

[+] sudo apt install ruby

[+] gem install lolcat

[+] git clone https://github.com/chill-c0der/geekstall.git

[+] cd geekstall

[+] sudo bash geekstall.sh | lolcat

TERMUX:

[+] apt upgrade
